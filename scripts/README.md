# Scripts

This directory contains utility scripts for the project.

## Available Scripts

- `build-safe.js` - Safe build process
- `bump-version.js` - Version bumping utility
- `create-release-zip.js` - Creates release zip files
- `release.js` - Release automation
- `sync-env-build.js` - Environment synchronization for builds

## Note

The page generator script has been removed. If you need to generate new pages, please create them manually following the existing patterns in the codebase.
